# Password Manager

## Project Documentation

This Password Manager allows users to securely store and manage their passwords. It uses strong encryption algorithms to ensure that your data remains safe.

### Features
- Store multiple passwords securely
- Generate strong passwords
- Auto-fill passwords in browsers
- Import/export passwords in various formats

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/Ashwini-Gunjal/password-manager.git
   ```

2. Navigate to the project directory:
   ```
   cd password-manager
   ```

3. Install the required dependencies:
   ```
   npm install
   ```

4. Create a `.env` file in the root directory and add your configuration settings:
   ```
   DATABASE_URL='your_database_url'
   ENCRYPTION_KEY='your_encryption_key'
   ```

5. Start the application:
   ```
   npm start
   ```

## Usage

- To add a new password, use the `add` command.
- To view your stored passwords, use the `view` command.
